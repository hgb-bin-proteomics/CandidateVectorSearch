Calculates the top n candidates for each spectrum on the CPU using Eigen.

Parameters
candidatesValuesAn integer array of theoretical ion m/z values for all candidates flattened.
candidatesIdxAn integer array that contains indices indicating where each candidate starts in candidatesValues.
spectraValuesAn integer array of peak m/z values from experimental spectra flattened.
spectraIdxAn integer array that contains indices indicating where each spectrum starts in spectraValues.
topNThe number (int) of top candidates that should be returned for each spectrum.
toleranceTolerance used for matching peaks in Dalton (float).
normalizeWhether or not the candidate scores should be normalized by candidate length (bool).
useGaussianTolWhether or not experimental peaks should be modelled as gaussian normal distributions (bool).
batchSizeIf a batched approach is used, how big should batches be (integer).
methodWhich matrix multiplication method should be used. See enum CPU_METHODS.
coresThe number of CPU cores that should be used for computation (int).
verboseAn integer parameter controlling how often progress should be printed to std::out. If 0 no progress will be printed.
memStatAn integer out parameter indicating if memory was successfully freed after execution, 0 = success, 1 = error.
Returns
An integer array with length (number of spectra * topN) containing the indices of the top n candidates for every spectrum.
